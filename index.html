Personal SMS Service Project

This page describes the opt-in process for Fernando Torres’ personal SMS learning project using Azure Communication Services. This is for development and testing only — not commercial use.

Questions? Contact contact@sms-service-project.org.

Program Description

Participants may receive occasional SMS test messages while I develop applications. Recipients include myself and a small group of friends who have provided direct consent.

Example Messages

  • [Azure SMS Test] Thank you for joining my personal SMS test program. Reply STOP to opt out.
  • [Azure SMS Test] This is a sample notification for testing purposes.
  • [Azure SMS Test] Debug: App triggered SMS successfully at 3:00 PM.

Message frequency is expected to be very low (typically under 10 per month). Message & data rates may apply. Carriers are not liable for delayed or undelivered messages.
